# memo-plugin-editg
editg is [mattn/memo](https://github.com/mattn/memo) plugin.  
Edit(Open editor) the line selected by grep.

## Installation
```
$ go get -u -d github.com/humangas/memo-plugin-editg
$ cd $GOPATH/src/github.com/humangas/memo-plugin-editg
$ make install
```

## Usage
```
$ memo editg
```
